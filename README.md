## 使用文档

- [x] **[😎 MINE-H5-UI@1.x 基于 Vue2.x](https://mineh5ui.biaov.cn/)**

## 安装

```Basic
npm i mine-h5-ui@version1
```

### 使用

```JavaScript
import Vue from 'vue'
import MINEH5UI from 'mine-h5-ui'
import 'mine-h5-ui/styles/index.css'

Vue.use(MINEH5UI)
```

- 👉 在[快速开始](https://mineh5ui.biaov.cn/doc/start)中查看更多信息。
