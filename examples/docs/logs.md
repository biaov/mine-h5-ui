# 更新日志

----

## 介绍

`MINE-H5-UI` 遵循 [Semver](https://semver.org/lang/zh-CN/) 语义化版本规范。

### 发布节奏

* 修订号：不定期发布，包含新特性和问题修复。
* 次版本号：不定期发布，包含新特性和较大的功能更新，向下兼容。
* 主版本号：发布时间不定，包含不兼容更新，预计作者技术发生重大提升或重构时发布。
