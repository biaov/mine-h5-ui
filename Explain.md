# 项目文档

## 运行项目

### 克隆项目

```Basic
git clone https://github.com/biaov/MINE-H5-UI.git
```

### 安装依赖

* 项目根目录安装项目依赖

```Basic
npm install
```

* 样式根目录安装样式依赖，路径：`/packages/theme-default`

```Basic
npm install
```

### 运行项目

```Basic
npm start
```

## 打包项目

```Basic
npm run build
```
